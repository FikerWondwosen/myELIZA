# myELIZA
This project is a small system in C or C++ that will simulate a primitive AI chatbot. Specifically, this will be implementing a version of ELIZA that is one of the earliest natural language processing programs. The program will simulate a conversation between the user and a psychiatrist.

___________________________________________

How to run the code:

type the following in the terminal for your dedicated system

Windows:

g++ .\main.cpp .\conversation.cpp

./a.exe


Linux CSC Machine:

g++ main.cpp conversation.cpp

./a.out
